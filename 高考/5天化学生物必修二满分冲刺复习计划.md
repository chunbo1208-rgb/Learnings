# 5天化学必修二 + 生物必修二满分冲刺复习计划

> 目标：用 5 天、每天约 2 小时，完成高中化学必修二和高中生物必修二的高强度查漏补缺。  
> 核心策略：先搭知识骨架，再练高频题型，最后用错题反推教材和概念。  
> 提醒：满分不是靠“看完书”，而是靠“概念零漏洞 + 题型有模板 + 错因不过夜”。

## 一、总复习原则

### 1. 每天 2 小时固定结构

每一天都按下面节奏执行：

| 时间 | 任务 | 目标 |
|---|---|---|
| 0-10 min | 回忆昨日错题和知识框架 | 激活记忆，防止假熟悉 |
| 10-45 min | 化学核心知识复习 | 抓反应、性质、原理、方程式 |
| 45-70 min | 化学题型训练 | 练选择题、推断题、实验题 |
| 70-105 min | 生物核心知识复习 | 抓概念、过程、规律、图像 |
| 105-120 min | 生物题型训练 + 错题整理 | 当天问题当天收口 |

### 2. 每天必须产出 3 个东西

1. 一页知识框架：只写关键词、箭头、易混点。
2. 一组错题记录：记录“错因”，不是抄题。
3. 一张满分检查清单：当天内容能不能不看书讲出来。

### 3. 错题记录格式

建议每道错题只写 4 行：

```text
题目主题：
我错在哪里：
正确判断依据：
下次看到类似题先看：
```

例如：

```text
题目主题：原电池电极判断
我错在哪里：只看金属活泼性，没有看反应本质
正确判断依据：负极发生氧化反应，电子从负极流出
下次看到类似题先看：谁失电子，谁是负极
```

## 三、Day 1：搭骨架，先把两科“地图”画出来

### 化学：元素周期律、化学键、物质结构

复习重点：

- 原子结构：质子数、核电荷数、核外电子排布。
- 元素周期表：周期、族、主族元素位置判断。
- 元素周期律：原子半径、金属性、非金属性、最高价氧化物水化物酸碱性、气态氢化物稳定性。
- 化学键：离子键、共价键、离子化合物、共价化合物。
- 电子式：简单离子化合物和共价分子的书写。

必须会背的规律：

- 同周期从左到右：原子半径减小，金属性减弱，非金属性增强。
- 同主族从上到下：原子半径增大，金属性增强，非金属性减弱。
- 非金属性越强：气态氢化物越稳定，最高价氧化物对应水化物酸性越强。
- 金属性越强：最高价氧化物对应水化物碱性越强。

训练题型：

- 根据元素位置推性质。
- 根据原子序数判断周期和族。
- 比较半径、金属性、非金属性。
- 判断化合物类型和化学键类型。
- 写电子式。

满分检查：

- 能不能 30 秒判断某元素在周期表中的位置？
- 能不能说清楚“半径、金属性、非金属性”的变化原因？
- 能不能区分“含离子键的一定是离子化合物”和“共价化合物只含共价键”？

### 生物：减数分裂、受精作用

复习重点：

- 减数分裂的本质：染色体复制一次，细胞连续分裂两次，染色体数目减半。
- 同源染色体、联会、四分体、交叉互换。
- 减数第一次分裂：同源染色体分离。
- 减数第二次分裂：姐妹染色单体分离。
- 精子和卵细胞形成过程差异。
- 受精作用：雌雄配子结合，恢复体细胞染色体数目。

必须会画的图：

- 一个 2n = 4 的细胞完成减数分裂全过程。
- 染色体数、DNA 数、染色单体数变化曲线。

训练题型：

- 看图判断分裂时期。
- 判断细胞名称：初级精母细胞、次级精母细胞、精细胞等。
- 判断染色体、DNA、染色单体数量。
- 判断异常配子形成原因。

满分检查：

- 能不能准确区分有丝分裂和减数分裂图像？
- 能不能看到“同源染色体分离”立刻想到减数第一次分裂？
- 能不能解释为什么子代既像父方又像母方？

## 四、Day 2：攻克计算和比例

### 化学：化学反应与能量、原电池

复习重点：

- 化学反应中能量变化：放热反应、吸热反应。
- 化学键与能量：断键吸热，成键放热。
- 常见放热反应：燃烧、中和、金属与酸反应、大多数化合反应。
- 常见吸热反应：Ba(OH)2·8H2O 与 NH4Cl 反应、C 与 CO2、C 与 H2O。
- 原电池条件：自发氧化还原反应、两个电极、电解质溶液、闭合回路。
- 原电池判断：负极氧化，正极还原；电子从负极流出，经导线到正极。

高频模板：

```text
判断原电池电极：
1. 先写总反应或找氧化还原关系
2. 失电子的是负极，发生氧化反应
3. 得电子的是正极，发生还原反应
4. 电子方向：负极 -> 导线 -> 正极
5. 电流方向：正极 -> 导线 -> 负极
```

训练题型：

- 能量图判断吸热、放热。
- 根据键能估算反应热。
- 原电池电极判断。
- 电极反应式书写。
- 金属腐蚀与防护。

满分检查：

- 能不能不用“活泼金属一定作负极”这种半截规律？
- 能不能准确写出 Zn-Cu 原电池中负极、正极、电极反应式？
- 能不能解释为什么原电池能把化学能转化为电能？

### 生物：孟德尔遗传规律

复习重点：

- 相对性状、显性性状、隐性性状、等位基因。
- 分离定律：等位基因随同源染色体分离而分离。
- 自由组合定律：非同源染色体上的非等位基因自由组合。
- 测交：用隐性纯合子检测未知显性个体基因型。
- 常见比例：3:1、1:1、9:3:3:1、1:1:1:1。

遗传题步骤：

```text
1. 先判断显隐性
2. 写亲本表现型和可能基因型
3. 写配子类型和比例
4. 画棋盘格或用分离组合法
5. 回到题目问法，答表现型比例或基因型比例
```

高频比例速记：

| 杂交类型 | 后代表现型比例 |
|---|---|
| Aa × Aa | 3:1 |
| Aa × aa | 1:1 |
| AaBb × AaBb | 9:3:3:1 |
| AaBb × aabb | 1:1:1:1 |
| AaBb × Aabb | 3:3:1:1 |

训练题型：

- 根据后代比例推亲本基因型。
- 测交实验分析。
- 两对相对性状自由组合。
- 致死、从性、环境影响类变式题。

满分检查：

- 能不能看到 9:3:3:1 立刻判断双杂合自交？
- 能不能把两对性状拆成两个一对性状分别算？
- 能不能区分表现型比例和基因型比例？

## 五、Day 3：拿下推断题核心

### 化学：有机物基础

复习重点：

- 有机物特点：多数含碳，结构复杂，反应慢，副反应多。
- 甲烷：取代反应。
- 乙烯：加成反应、使溴水或酸性高锰酸钾褪色。
- 苯：结构特殊，易取代，难加成。
- 乙醇：与钠反应、氧化反应。
- 乙酸：酸性、酯化反应。
- 酯化反应特点：酸脱羟基，醇脱氢；浓硫酸作催化剂和吸水剂。

必背反应：

```text
CH4 + Cl2 -> CH3Cl + HCl（光照，取代）
CH2=CH2 + Br2 -> CH2Br-CH2Br（加成）
2CH3CH2OH + 2Na -> 2CH3CH2ONa + H2
2CH3CH2OH + O2 -> 2CH3CHO + 2H2O（Cu/Ag，加热）
CH3COOH + CH3CH2OH <=> CH3COOCH2CH3 + H2O（浓硫酸，加热）
```

训练题型：

- 根据官能团判断性质。
- 有机物鉴别。
- 有机推断流程。
- 酯化实验装置和现象。
- 同分异构体简单判断。

满分检查：

- 能不能看到碳碳双键立刻想到加成和褪色？
- 能不能解释乙醇、乙酸、乙酸乙酯的性质差异？
- 能不能把“取代、加成、氧化、酯化”区分清楚？

### 生物：伴性遗传、人类遗传病

复习重点：

- 性染色体：男性 XY，女性 XX。
- 伴 X 隐性遗传：男患者多，女患者父亲和儿子常有规律。
- 伴 X 显性遗传：女患者多，男患者的女儿往往患病。
- 伴 Y 遗传：父传子，子传孙，只在男性中出现。
- 人类遗传病类型：单基因遗传病、多基因遗传病、染色体异常遗传病。
- 遗传咨询和产前诊断。

系谱图判断顺序：

```text
1. 无中生有，多为隐性
2. 有中生无，多为显性
3. 看男女患病比例
4. 看是否父传子，判断是否可能伴 Y
5. 看女患者父亲/儿子，或男患者母亲/女儿，判断是否伴 X
6. 不能确定时，优先写“可能”
```

训练题型：

- 系谱图判断遗传方式。
- 计算后代患病概率。
- 判断携带者概率。
- 区分常染色体遗传和伴性遗传。

满分检查：

- 能不能准确使用“无中生有为隐性，有中生无为显性”？
- 能不能避免把所有男女比例不同都直接判成伴性遗传？
- 能不能算出“正常个体是携带者”的条件概率？

## 六、Day 4：综合提升，专攻实验、图像、表格

### 化学：资源开发、实验与综合推断

复习重点：

- 海水资源：海水制盐、溴、镁等。
- 金属矿物开发：金属冶炼方法与金属活动性关系。
- 化石燃料：煤、石油、天然气；石油分馏、裂化、裂解。
- 环境保护：绿色化学、原子利用率、污染控制。
- 实验题：操作目的、试剂作用、现象描述、误差分析。

实验题答题语言：

```text
检验气密性：
连接装置，将导管末端插入水中，微热容器，若导管口有气泡冒出，冷却后导管内形成一段水柱，说明气密性良好。

洗涤沉淀：
向过滤器中的沉淀加入蒸馏水，使水浸没沉淀，待水自然流下，重复 2-3 次。

判断沉淀洗净：
取最后一次洗涤液，加入相应检验试剂，若无特征现象，说明已洗净。
```

训练题型：

- 工艺流程题。
- 实验目的和试剂作用。
- 物质分离提纯。
- 方程式补全。
- 环保和绿色化学选择题。

满分检查：

- 能不能写出规范实验现象，而不是只写“有反应”？
- 能不能区分蒸馏、分馏、萃取、过滤、蒸发？
- 能不能在流程题中标出“加入物质的目的”？

### 生物：DNA、基因表达、变异与进化

复习重点：

- DNA 是主要遗传物质。
- DNA 双螺旋结构：两条链反向平行，碱基互补配对。
- DNA 复制：半保留复制，边解旋边复制。
- 基因指导蛋白质合成：转录、翻译。
- 中心法则：DNA -> RNA -> 蛋白质。
- 基因突变、基因重组、染色体变异。
- 现代生物进化理论：种群是进化基本单位，突变和基因重组提供原材料，自然选择决定方向，隔离导致物种形成。

易混点：

| 概念 | 本质 | 常考点 |
|---|---|---|
| 基因突变 | DNA 碱基序列改变 | 可产生新基因 |
| 基因重组 | 原有基因重新组合 | 不能产生新基因 |
| 染色体结构变异 | 片段缺失、重复、倒位、易位 | 显微镜下可能观察 |
| 染色体数目变异 | 染色体数变化 | 单体、三体、多倍体 |

训练题型：

- DNA 复制相关计算。
- 碱基互补配对计算。
- 转录和翻译过程判断。
- 密码子、反密码子、氨基酸关系。
- 变异类型判断。
- 进化观点判断。

满分检查：

- 能不能区分复制、转录、翻译的模板、原料、产物、场所？
- 能不能准确判断“可遗传变异”和“不可遗传变异”？
- 能不能解释自然选择如何改变种群基因频率？

## 八、化学必修二终极清单

### 必会知识

- 元素周期表结构：周期、族、主族。
- 元素周期律：半径、金属性、非金属性、酸碱性、氢化物稳定性。
- 化学键：离子键、共价键、离子化合物、共价化合物。
- 化学反应与能量：吸热、放热、断键吸热、成键放热。
- 原电池：负极氧化，正极还原，电子从负极流向正极。
- 甲烷取代、乙烯加成、乙醇氧化、乙酸酯化。
- 海水资源、金属冶炼、石油分馏和裂化。
- 绿色化学和环境保护。

### 必会方程式

```text
2Na + 2H2O = 2NaOH + H2
Cl2 + H2O = HCl + HClO
SO2 + H2O <=> H2SO3
3NO2 + H2O = 2HNO3 + NO
CH4 + Cl2 -> CH3Cl + HCl（光照）
CH2=CH2 + Br2 -> CH2BrCH2Br
2CH3CH2OH + 2Na -> 2CH3CH2ONa + H2
2CH3CH2OH + O2 -> 2CH3CHO + 2H2O（Cu/Ag，加热）
CH3COOH + CH3CH2OH <=> CH3COOCH2CH3 + H2O
```

### 化学考场策略

- 选择题先排除明显错误，再回到规律。
- 原电池题先找氧化还原，不要只看金属活泼性。
- 有机题先圈官能团，再判断反应类型。
- 实验题回答“操作 + 现象 + 结论”，不要只写结论。
- 流程题每加入一种试剂，都问一句：除杂、氧化、还原、沉淀、调 pH，还是转化？

## 九、生物必修二终极清单

### 必会知识

- 减数分裂：同源染色体分离发生在减数第一次分裂。
- 受精作用：恢复染色体数目，增加遗传多样性。
- 分离定律：等位基因分离。
- 自由组合定律：非同源染色体上的非等位基因自由组合。
- 伴性遗传：重点掌握伴 X 隐性、伴 X 显性、伴 Y。
- DNA 双螺旋结构和碱基互补配对。
- DNA 复制：半保留复制。
- 基因表达：转录和翻译。
- 变异：基因突变、基因重组、染色体变异。
- 进化：种群、基因频率、自然选择、隔离、物种形成。

### 生物高频判断句

```text
基因突变可以产生新基因。
基因重组不能产生新基因，但能产生新的基因型。
自然选择直接作用于表现型。
生物进化的实质是种群基因频率的改变。
隔离是物种形成的必要条件。
DNA 复制需要模板、原料、能量和酶。
密码子位于 mRNA 上，反密码子位于 tRNA 上。
```

### 生物考场策略

- 遗传题先写基因型，不要只凭感觉看表现型。
- 系谱图先判断显隐性，再判断常染色体或伴性遗传。
- 图像题先看横纵坐标和单位。
- 选择题遇到“都、一定、完全、只”等绝对词要格外谨慎。
- 计算概率时，先算每一个独立事件，再相乘。

## 十二、最后提醒

这 5 天不要追求“资料很多”，只追求“每个高频点都能拿分”。  
复习顺序永远是：

```text
教材概念 -> 高频规律 -> 典型题 -> 错题复盘 -> 闭卷复述
```

如果某一章你觉得自己“看懂了”，但做题错得多，说明它还没有变成考场能力。  
真正的满分状态是：看到题目能立刻判断它在考哪个概念、用哪个规律、避开哪个坑。
