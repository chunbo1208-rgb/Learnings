# Problem Statement
A set of positive integers is called **fragrant** if it contains at least two elements and each element shares a common prime factor with at least one other element of the set. Let $P(n)=n^2+n+1$. Determine the smallest positive integer $b$ such that there exists a nonnegative integer $a$ for which the set
$$\{P(a+1),P(a+2),\dots,P(a+b)\}$$
is fragrant.
