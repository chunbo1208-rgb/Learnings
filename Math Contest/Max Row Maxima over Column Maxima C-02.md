# Problem Statement
In a $2025\times 2025$ board, each cell is filled with a nonnegative real number so that every row sum and every column sum is $1$. Let $R$ be the sum of the largest entry in each row, and let $C$ be the sum of the largest entry in each column. Determine the maximum possible value of $R/C$.
