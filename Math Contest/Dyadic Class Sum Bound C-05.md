# Problem Statement
Let $n,k$ be positive integers, and let $a_i\in[1,2^k]$ for $1\le i\le n$. Prove that
$$\sum_{i=1}^n \frac{\sqrt{a_i}}{a_1^2+\cdots+a_i^2}\le 4\sqrt{kn}.$$
