# Problem Statement
There are $n$ types of coins, and the value of the $i$-th type is $d_i$, where $d_1,d_2,\dots,d_n$ are distinct positive integers. A positive integer $D$ is called **lucky** if for every pile of coins, whenever the total value of the pile is divisible by $D$, the pile can be partitioned into groups each having total value exactly $D$. Must a lucky integer always exist?
