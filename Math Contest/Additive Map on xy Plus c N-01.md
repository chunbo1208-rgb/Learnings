# Problem Statement
Let $S$ be the set of all positive integers not smaller than $10^{100}$, and let $c$ be a given positive integer. Determine all maps $f:S\to\mathbb Z$ such that
$$f(xy+c)=f(x)+f(y)$$
for all $x,y\in S$.
