# Problem Statement
Let $S$ be a set of positive integers satisfying:

1. $1\in S$;
2. For every integer $n>1$, one has $n\in S$ if and only if
$$\sum_{\substack{d\mid n,\ d<n \\ d\in S}} d\le n.$$

If equality holds in the displayed condition for some $n\in S$, then $n$ is called a **good number**. Determine the necessary and sufficient conditions on a positive integer $m$ and an odd prime $p$ such that $2^m p$ is a good number.
