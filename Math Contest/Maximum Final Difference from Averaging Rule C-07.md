# Problem Statement
Let $a_0,a_1,\dots$ be a real sequence with $a_0=0$ and $a_1=1$. For each $n\ge 2$, assume there exists an integer $k$ with $1\le k\le n$ such that
$$a_n=\frac{a_{n-1}+\cdots+a_{n-k}}{k}.$$
Determine the maximum possible value of $a_{2018}-a_{2017}$.
