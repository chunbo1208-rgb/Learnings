# Problem Statement
Given a positive integer $k$, determine all functions $f:\mathbb Z_{>0}\to\mathbb Z_{>0}$ such that for all positive integers $x,y$,
$$\bigl(f(x)+f(y)\bigr)^2-4kxy$$
is a perfect square.
