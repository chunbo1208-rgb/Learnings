# What is Vertically Learning?
Vertically learning is a new method of learning because of the development of AI. AI is the character of "well known man".

The learner will start with scratch (no more knowledge) but directly involved in a whole project or a problem, and goal oriented, learn the logic and language along the way.
# Three main ways of vertically learning
## On Subject Learning
For a certain subject we start with new and directly ask some question
- Why I want to learn this? (Interesting? Useful? Exam?)
- What is the main problem this subject talking about?
- What's the top result in this area? List the famous and important result (as essays) in this.

Then focus on a result, I will do it on myself again and understand all the details, then understanding the main logic of solving problems.

We will try to dive deeper or innovation in some part, or simplify the project.

## On Engineering
For example, we will redo a Github Repo.

ask:
- What is this project mainly do, and how does it achieve?

Then we need to dive deeper through it.
